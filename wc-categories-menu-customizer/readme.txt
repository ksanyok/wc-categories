== Description ==
Quickly organize WooCommerce product categories into menus with this easy-to-use menu customizer. 

This plugin allows for simple creation and updating of product categories menus directly from the WordPress admin dashboard. If managing a multitude of WooCommerce categories is a challenge, WC Categories Menu Customizer simplifies the process by auto-populating your menu with WooCommerce product categories, maintaining the proper hierarchical structure. This ensures your products are presented beautifully and intuitively, saving you time and hassle.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/woocommerce-categories-menu-creator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the admin menu page to create or update your categories menu.

== Frequently Asked Questions ==
= How do I create a menu? =
After activation, navigate to the plugin's menu in the admin dashboard and enter a menu name to create or update.
